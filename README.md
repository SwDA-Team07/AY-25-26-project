# Software Design and Architecture Project

**A.Y. 2025/2026**

## Project Overview

This repository contains the project work for the *Software Design and Architecture* course. The selected open-source system is **Apache Log4j2**.

The analysis covers:

* **Software Design** aspects (dependencies, design patterns)
* **Software Architecture** aspects (using the C4 model)

---

## Analyzed System

* **Name:** Apache Log4j2
* **Type:** Java-based logging framework
* **Repository:** https://github.com/apache/logging-log4j2
* **Reason for selection:**
  The system is widely used, actively maintained, and sufficiently complex (~100k+ LOC), making it suitable for architectural and design analysis.
