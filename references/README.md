# References

This directory contains external resources used in the project.

These may include:

* Official documentation of the analyzed system
* Relevant technical resources
* Academic papers and publications

All references support the content presented in the reports.

---

## How to Add Links

When adding a link to [links.md](links.md):

1. Choose an appropriate category (or create a new one)
2. Add the entry in this format:
   ```
   - **[Resource Name]:** [URL]
     - Brief description (one line)
   ```
3. Example:
   ```
   - **Apache Log4j2 GitHub:** https://github.com/apache/logging-log4j2
     - Official repository for Apache Log4j2 framework
   ```

## How to Add Papers

When adding a paper to [papers.md](papers.md):

1. Choose an appropriate category (or create a new one)
2. Add the entry in this format:
   ```
   - **Author(s) (Year).** *Title.* Publisher/Conference.
     - Brief description (one-two lines)
   ```
3. Example:
   ```
   - **Gamma, E., Helm, R., Johnson, R., & Vlissides, J. (1994).** *Design Patterns: Elements of Reusable Object-Oriented Software.* Addison-Wesley.
     - Classic reference for Gang of Four design patterns
   ```
