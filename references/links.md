# External Links

Useful online resources related to the Apache Log4j2 project analysis.

## Categories

### Official Documentation

- **Apache Log4j2 Repository:** https://github.com/apache/logging-log4j2
  - Source system analyzed in this project (branch `2.x`, fixed snapshot for reproducibility).

- **Apache Log4j2 Architecture Manual:** https://github.com/apache/logging-log4j2/blob/2.x/src/site/antora/modules/ROOT/pages/manual/architecture.adoc
  - Upstream architectural documentation used as contextual reference (not a substitute for course C4 deliverables).

### C4 Model

- **Structurizr DSL Documentation:** https://docs.structurizr.com/dsl
  - Official Structurizr DSL documentation used for the C4 workspace model in `diagrams/architecture/workspace.dsl`.

- **Structurizr CLI Export Command:** https://docs.structurizr.com/cli/export
  - Official CLI export reference used to export the DSL workspace directly to SVG diagrams.

- **Mermaid C4 Diagram Syntax:** https://mermaid.ai/open-source/syntax/c4.html
  - Historical reference for the previous C4 diagram implementation before migration to Structurizr.

- **Mermaid Documentation:** https://mermaid.js.org/
  - Diagramming syntax still used for non-C4 diagrams in other reports, such as Overview and Design.

### Code Analysis Tools

- **Git rev-list:** https://git-scm.com/docs/git-rev-list
  - Used to enumerate commits in the co-change analysis window.

- **Git show:** https://git-scm.com/docs/git-show
  - Used to extract per-commit changed files for pairwise co-change counting.

- **Python csv module:** https://docs.python.org/3/library/csv.html
  - Used to generate reproducible CSV artifacts for dependency evidence.
